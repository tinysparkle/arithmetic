1. 遍历单词库数组
2. indexOf判断前缀

1. 提前把单词库拆分成26个数组
2. 第一层， 第二层， 第三层 ...最后一层拆分成对象
3. 最后变成一颗单词树

{
  a: [...],
  b: [...],
  c: {
    a: [...],
    b: [...],
    c: [...],
    ...
  },
  ...
}
